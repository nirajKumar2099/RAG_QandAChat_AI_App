Q: How does your RAG pipeline work?

Answer:

The /ask-question API orchestrates the flow.
It generates the query embedding, calls the /search-context API 
to retrieve relevant documents using pgvector similarity search,
and then sends the retrieved context along with the user query to the 
LLM to generate the final response.
